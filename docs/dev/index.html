===Event Handlers=== BattleArena uses a custom annotation, '''@ArenaEventHandler''', that allows you to only listen for events that occur inside an Arena. This is used like the normal '''@EventHandler''' annotation but greatly simplifies code as all the checking to make sure a player is in a game, and finding the correct game, is already taken care of. @ArenaEventHandler: @ArenaEventHandler public void onBlockPlace(BlockPlaceEvent event){ /// This event will only be called for players inside the arena placing blocks }
